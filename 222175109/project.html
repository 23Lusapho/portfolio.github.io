About the Project

SciTechAdv is a web-based Student Management System developed as part of the Software Engineering II course. The system enhances educational management by automating registration, grading, and communication processes between lecturers and students. It focuses on efficiency, security, and modern interface design, ensuring a seamless user experience for all users.

Project Objectives

  • To design a secure and user-friendly academic data management system.
  • To streamline marks entry, student registration, and progress tracking.
  • To ensure reliable and accurate database integration with MySQL.
  • To provide separate user roles for Admin, Lecturer, and Student.
  • To deliver an intuitive, responsive, and dynamic interface using PHP and JSP.
